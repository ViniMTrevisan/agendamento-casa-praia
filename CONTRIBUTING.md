# Como Contribuir

Agradecemos o interesse em contribuir!

## Reportando Bugs

Se você encontrar um bug, por favor, abra uma "Issue" (Problema) e inclua:
1.  Passos para reproduzir o bug.
2.  O que você esperava que acontecesse.
3.  O que realmente aconteceu (com screenshots e logs, se possível).

## Sugerindo Novas Features

Abra uma "Issue" descrevendo a sua ideia e o valor que ela agregaria à aplicação.

## Fluxo de Pull Request (PR)

1.  Faça um "fork" do repositório.
2.  Crie uma nova branch para a sua feature (`git checkout -b feat/minha-feature`).
3.  Faça o commit das suas mudanças (`git commit -m "feat: adiciona minha feature"`).
4.  Certifique-se de que os testes passam (`npm run test:unit`).
5.  Faça o push para a sua branch (`git push origin feat/minha-feature`).
6.  Abra um Pull Request para a branch `main`.
